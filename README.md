# assistantOne

News aggregator, with aspiration of broader assistant abilities to come.

## Overview

assistantOne is a small project that aggregates news from configurable sources and provides a foundation for building broader assistant capabilities on top of that data.

## Features

- Fetch and aggregate news from multiple sources
- Normalize and store article metadata
- Simple command-line and/or web interface (TBD)
